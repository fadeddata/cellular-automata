Cellular Automata loosely based on rot.js (http://ondras.github.io/rot.js/manual/#map/cellular)

Quick Example
=============

```
sbt
> console
scala> import dungeon.utils._
scala> dungeon.Cellular(80,25, born = Seq(5,6,8)).iterate(20, Some(render _))
```
